Atividade Questionario

Questões HTML e CSS

  1. Como comentar tags HTML?

    2. R:Para adicionar um comentário em um código HTML, é preciso utilizar as tags próprias para esse propósito. Portanto, todo o texto que estiver delimitado pelas tags >! –“, que indica o início do trecho, e “–>” que significa o fechamento, é considerado comentário

  2. criar um hiperlink?

    3. R:Para criar um link em uma página da web, você precisa envolver um elemento (um texto, uma imagem e assim por diante) em um elemento de âncora ( < a > ) e definir seu atributo href para que aponte para o URL o qual você quer associar

  3. Como criar uma abreviação ou um acrônimo?

    4. R:O Elemento _HTML < abbr > _(ou Elemento de Abreviação HTML) representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela

  4. Qual é a maneira correta de escrever o endereço em um documento HTML?

    5. R:A tag < address > define as informações de contato do autor / proprietário de um documento ou um artigo. Se o elemento < address > está dentro do elemento < body >, representa informações de contato para o documento.

  5. Como criar uma área dentro de um mapa de imagem?

    6. R:A abordagem deste artigo é criar uma área dentro de um mapa de imagem usando uma tag < area > em um documento HTML. É usado para direcionar o usuário a diferentes links depois que o usuário clica nas partes mapeadas da imagem. É usado como uma tag filha da tag < map >. Não tem tag de fechamento e, portanto, é uma tag vazia para HTML, mas no caso de XHTML, você precisa fechá-la corretamente.

  6. Criar um documento HTML que usa o elemento artigo?

    7. R:O Elemento HTML Article (< article >) representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuido de forma independente ou reutilizável, por exemplo, em sindicação

  7. Criar um documento HTML que usa o elemento side?

    8. R:O elemento HTML < aside >representa uma seção de uma página que consiste em um conteúdo tangencialmente relacionado ao conteúdo de seu ambiente, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais. Elas muitas vezes contemplam divergências laterais, como a definição de um glossário; conteúdo vagamente relacionado, como avisos; uma biografia do autor; ou, em aplicações web, informações de perfil ou links de blogs relacionados.

  8. Como incorporar áudio em um documento HTML?

    9. R:O controls atributo adiciona controles de áudio, como reprodução, pausa e volume. O < source >elemento permite que você especifique arquivos de áudio alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido. O texto entre as tags < audio >e < /audio >só será exibido em navegadores que não suportam o < audio >elemento.

  9. Como escrever texto em negrito usando tags HTML?

    10. R:Utilizando a ta "b" antes do texto pretendido para ficar negrito

  10. Como especificar a URL/alvo base para todas as URLs relativas em um documento?

    11. R:Uma URL relativa localiza um recurso usando uma URL absoluta como ponto de partida. Na verdade, a "URL completa" do destino é especificada pela concatenação das URLs absolutas e relativas. Normalmente, uma URL relativa consiste apenas no caminho e, opcionalmente, no recurso, mas em nenhum esquema nem servidor
